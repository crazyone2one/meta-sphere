# Spring Boot 3 + Spring Security框架

一个基于 Spring Boot 3.x 、Spring Security、vue 3.x 的框架。

## 🚀 特性

- [x] Spring Boot 3 (Jakarta EE)
- [ ] 权限管理
- [ ] 接口测试
- [ ] 自动化测试
- [ ] 压力测试测试
- [ ] JMeter 5 非 GUI 自动执行
- [ ] YAML/Excel 数据驱动
- [ ] 自定义 Java 断言
- [ ] HTML 报告生成
- [ ] CI/CD 友好

## 🛠 环境要求

- JDK 21+
- Apache JMeter 5.6+
- Maven 3.9+

## ⚙️ 配置

1. 安装 JMeter 并设置 `test.jmeter.home` 路径
2. 修改 `application-[env].yml` 中的 `base-url` 和路径

## ▶️ 运行

```bash
mvn spring-boot:run
